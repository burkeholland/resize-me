ResizeMe Features and FAQ

Everything you need to know about ResizeMe for macOS and Windows.

Features

  • One global hotkey to resize active windows quickly.
  • Preset window dimensions for common workflows and screen sizes.
  • Support for both macOS and Windows.
  • Fast Windows install and upgrades through winget.
  • Homebrew install and upgrade path for the macOS app.

Frequently asked questions

Is ResizeMe free?

Yes. ResizeMe is available as a free install through winget on Windows, Homebrew on macOS, and GitHub Releases for manual downloads.

Does ResizeMe work on macOS and Windows?

Yes. The project ships builds for both platforms.

How do I install on Windows?

Use winget install BurkeHolland.ResizeMe. Update later with winget upgrade BurkeHolland.ResizeMe.

How do I install on macOS?

Use the Homebrew install commands shown on the homepage, or download manually from Releases.

Where do I get the latest version?

Use the latest release page at GitHub Releases.
